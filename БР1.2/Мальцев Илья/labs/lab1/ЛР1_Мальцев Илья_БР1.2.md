## Цель работы

Реализовать работающее REST API по ранее спроектированной OpenAPI-схеме для сайта поиска работы.

## Реализованная функциональность

- регистрация и вход пользователя;
- получение текущего пользователя по Bearer-токену;
- справочники навыков и отраслей;
- поиск вакансий с фильтрами;
- получение детальной информации о вакансии;
- создание резюме соискателя;
- просмотр резюме и откликов соискателя;
- создание отклика на вакансию;
- просмотр, создание, обновление и закрытие вакансий работодателя;
- изменение статуса отклика работодателем.

## Архитектура

Проект реализован на Go без внешних зависимостей.

- `cmd/job-search-api` - точка входа приложения;
- `internal/app` - маршрутизация HTTP-запросов;
- `internal/controllers` - контроллеры REST API;
- `internal/models` - доменные модели;
- `internal/views` - представления ответов API;
- `internal/store` - in-memory хранилище и бизнес-операции;
- `internal/httpx` - JSON-ответы, ошибки и CORS;
- `internal/errs` - единый формат ошибок.
