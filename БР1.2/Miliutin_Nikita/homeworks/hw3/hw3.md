ДЗ3: Тестирование API средствами Postman

Срок: 15.04.26

Задание:

    реализовать тестирование API средствами Postman;
    написать тесты внутри Postman.

В результате хочу получить один комплексный сценарий по основному процессу в вашем приложении.

Пример такого сценарий для системы по аренде недвижимости: регистрация -> вход -> список объявлений -> фильтрация по списку -> просмотр случайного объявления из списка -> старт переписки по объявлению.
