🔐 ЭНДПОИНТЫ - 37 ВСЕГО
────────────────────────────────────────────────────────────────────────────

Аутентификация (4):
  POST   /auth/register              - Регистрация нового пользователя
  POST   /auth/login                 - Вход в систему
  POST   /auth/refresh               - Обновление access токена
  POST   /auth/logout                - Выход из системы

Профиль пользователя (3):
  GET    /users/me                   - Получить свой профиль
  PATCH  /users/me                   - Обновить профиль
  PUT    /users/me/password          - Сменить пароль

Вакансии (8):
  GET    /vacancies                  - Поиск с фильтрацией
  POST   /vacancies                  - Создать вакансию
  GET    /vacancies/{id}             - Получить вакансию
  PUT    /vacancies/{id}             - Обновить вакансию
  DELETE /vacancies/{id}             - Удалить вакансию
  GET    /vacancies/my               - Мои вакансии (работодатель)
  GET    /vacancies/{id}/applications - Отклики на вакансию
  POST   /vacancies/{id}/apply       - Откликнуться

Резюме (11):
  GET    /resumes                    - Список резюме
  POST   /resumes                    - Создать резюме
  GET    /resumes/{id}               - Получить резюме
  PUT    /resumes/{id}               - Обновить резюме
  DELETE /resumes/{id}               - Удалить резюме
  POST   /resumes/{id}/experience    - Добавить опыт
  PUT    /resumes/{id}/experience/{exp_id}   - Обновить опыт
  DELETE /resumes/{id}/experience/{exp_id}   - Удалить опыт
  POST   /resumes/{id}/education     - Добавить образование
  PUT    /resumes/{id}/education/{edu_id}    - Обновить образование
  DELETE /resumes/{id}/education/{edu_id}    - Удалить образование
  PUT    /resumes/{id}/skills        - Обновить навыки

Отклики (4):
  GET    /applications/my            - Мои отклики (соискатель)
  POST   /vacancies/{id}/apply       - Откликнуться на вакансию
  GET    /vacancies/{id}/applications - Отклики (работодатель)
  PATCH  /applications/{id}/status   - Изменить статус

Навыки (1):
  GET    /skills                     - Справочник навыков

✨ ФУНКЦИОНАЛЬНОСТЬ
────────────────────────────────────────────────────────────────────────────
Поиск и фильтрация:
  ✅ Полнотекстовый поиск по вакансиям (q parameter)
  ✅ Фильтр по городу, зарплате, опыту, типу занятости
  ✅ Фильтр по навыкам (many-to-many)
  ✅ Пагинация (page, per_page)

Аутентификация:
  ✅ JWT токены (access + refresh)
  ✅ Bcrypt хеширование паролей
  ✅ Bearer token в заголовках
  ✅ 30-минутный access token
  ✅ 7-дневный refresh token

Авторизация:
  ✅ Role-based access control (applicant/employer)
  ✅ Ownership checks (может редактировать только свои ресурсы)
  ✅ Правильные HTTP коды (401, 403)

Валидация:
  ✅ Pydantic для всех входных данных
  ✅ Type hints для всех параметров
  ✅ Проверка на уникальность email
  ✅ Проверка дублирования откликов

Отношения в БД:
  ✅ One-to-many (User → Resumes, Company → Vacancies)
  ✅ Many-to-many (Resume/Vacancy → Skills)
  ✅ Cascading deletes
  ✅ Foreign keys
