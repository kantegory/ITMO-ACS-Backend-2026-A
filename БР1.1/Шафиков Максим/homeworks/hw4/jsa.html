Job Search Platform — Микросервисная архитектура

Платформа поиска работы · Database-per-Service · Go / Gin / PostgreSQL

6 микросервисов

  • • Auth Service — регистрация, логин, JWT
  • • Profile Service — компании, профили
  • • Vacancy Service — вакансии, поиск
  • • Resume Service — резюме, навыки
  • • Application Service — отклики
  • • Dictionary Service — справочники

Database-per-Service

  • • 6 отдельных PostgreSQL инстансов
  • • Логические связи через UUID (без FK)
  • • Только Dictionary Service — read-only
  • • Auth + Dictionary — zero dependencies
  • • Application — самый связанный (3 upstream)

Взаимодействие

  • • Synchronous: Internal REST API
  • • Auth: Shared JWT Secret
  • • Internal: X-Internal-Token header
  • • Service discovery: Docker DNS
  • • Future: Message queue (RabbitMQ)
